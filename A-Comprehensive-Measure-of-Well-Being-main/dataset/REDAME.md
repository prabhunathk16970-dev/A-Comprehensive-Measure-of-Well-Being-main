# 🌍 Human Development Index (HDI) Predictor

## 📌 Project Overview

The Human Development Index (HDI) Predictor is a Machine Learning web application that predicts the Human Development Index of a country using important socio-economic indicators.

The application is built using Python, Flask, Pandas, NumPy, Scikit-learn, Matplotlib and Seaborn.

---

## Features

- Predict HDI Score
- Classify Human Development Level
- Interactive Web Application
- Data Visualization
- Machine Learning Model
- Responsive UI

---

## Technologies

- Python
- Flask
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- HTML
- CSS

---

## Machine Learning Algorithm

Linear Regression

---

## Input Parameters

- Life Expectancy
- Expected Years of Schooling
- Mean Years of Schooling
- Gross National Income (PPP)

---

## Output

- Predicted HDI Score
- Human Development Category

---

## Dataset

Official UNDP Human Development Dataset
